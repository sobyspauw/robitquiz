## Tips for Using with AI Image Generators

### For DALL-E 3 (ChatGPT Plus):
1. Upload 2-3 existing images (animals.png, technology.png, space & astronomy.png) as reference
2. Say: "Create a new image in exactly this style for the topic [TOPIC]"
3. Paste the prompt
4. If text is wrong, regenerate and specify "make sure the text '[TOPIC]' is clearly readable"

### For Ideogram 2.0:
1. Use "Image prompt" mode
2. Paste the full prompt
3. Enable "Typography" mode for better text
4. Use existing image as style reference if available

### For Midjourney:
1. Upload existing image and get image URL
2. Use prompt format: `[IMAGE_URL] [PROMPT] --sref [IMAGE_URL] --sw 1000 --ar 16:9`
3. The `--sref` parameter ensures style consistency
4. `--ar 16:9` creates wide horizontal format

### General Tips:
- Always upload 1-2 existing images as style reference
- Generate 2-3 variations per topic and pick best
- If robot looks different, emphasize "rectangular metallic body" and "yellow star eyes"
- If icons are too detailed, add "simple flat icon style"
- If background isn't transparent, specify "PNG with transparent background"
